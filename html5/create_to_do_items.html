Create to-do items

  1. Go to the Garden menu, and then tap the To-dos tab.
  2. To add a new item, tap +.
  3. Enter a Name for your to-do item.
  4. Optional: Specify time management settings for the to-do item:
    1. To set a Reminder to start focusing on an item, select the time and the days when you would like to receive notifications.
    2. Set a Goal for completing the to-do item.
      If you aim to complete this to-do item in 3 hours, select 3 h.
    3. Decide if you want to reset the goal on a Daily, Weekly, or Monthly basis.
      If you want to set aside 3 hours every day for this to-do item, select Daily.
    4. To track your habits, set Habit Tracker to Weekly or Monthly.
  5. Tap OK.
